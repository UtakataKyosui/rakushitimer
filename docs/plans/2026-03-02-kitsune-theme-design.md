# デザイン設計書：「きつね色の台所」テーマ

日付: 2026-03-02
対象: いなりといっしょ！アプリ UI テーマ刷新

---

## 背景・動機

- ライバー「楽下いなり」は狐キャラ＋関西の「おかん」的ポジション
- 「いなり」= いなり寿司の連想から **きつね色（油揚げ色）** をテーマカラーに
- 従来の赤＋黒は警戒色に見えるため、温かみのある黄金橙＋焦げ茶へ変更
- オカン感・いなり寿司感をカラー・テキスト・パターンで表現

---

## カラーパレット

| トークン | 値（oklch） | 用途 |
|----------|-------------|------|
| `--brand` | `oklch(0.65 0.17 55)` | きつね色（明るい琥珀・黄金橙） |
| `--brand-dark` | `oklch(0.22 0.05 42)` | 焦げ茶（醤油タレ色） |
| `--background` | `oklch(0.99 0.01 55)` | 温かみのあるクリーム白 |
| `--primary` | = `--brand` | ボタン・アクティブ状態 |
| `--accent` | `oklch(0.95 0.03 55)` | 薄い油揚げ色（ホバー背景等） |

---

## ビジュアル要素

### Navbar
- 背景: 焦げ茶→少し明るい焦げ茶のグラデーション（`--brand-dark` ベース）
- 薄い菱形パターン（CSS `background-image` で実装）
- テキスト: 白

### ボトムナビゲーション
- 背景: `bg-brand-dark`（焦げ茶）
- アクティブ: `text-brand`（きつね色でひかる）
- 非アクティブ: `text-white/60`

### アラームカード
- 背景: `oklch(0.97 0.01 55)`（和紙風の温かみある白）
- ボーダー: `oklch(0.90 0.03 55)`（薄い油揚げ色）
